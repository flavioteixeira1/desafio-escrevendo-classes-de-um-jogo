# desafio-escrevendo-classes-de-um-jogo
código em Java que descreve uma classe herói para um jogo de aventura

git clone https://github.com/flavioteixeira1/desafio-escrevendo-classes-de-um-jogo

cd desafio-escrevendo-classes-de-um-jogo

javac src/*.java

cd src

java Main


Neste exemplo são criados quatro heróis, que atacam de modo diferente,a depender do tipo
de herói que foi criado, conforme a classe hero.java e a enum caracther.java
